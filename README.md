# HYG Asesoría Legal

Sitio web para HYG Asesoría Legal — despacho de abogados en Durango, Durango, México.

Inspirado en la estructura de brazeauseller.com pero aplicando los principios de diseño codificados (escala tipográfica modular, retícula de 8 puntos, color 60-30-10, acento único por sección) desarrollados para el sitio de T.C Consulting.

## Stack

HTML + CSS + JS estáticos. Sin build. Se despliega directo a GitHub Pages.

## Archivos

- `index.html` — página única con todas las secciones
- `styles.css` — tokens de diseño y estilos
- `script.js` — nav fijo, menú móvil, año del footer
- `assets/` — imágenes (logo pendiente de proporcionar por el cliente)

## Vista previa local

```bash
npx http-server -p 8765 -c-1
```

Abrir http://127.0.0.1:8765/
